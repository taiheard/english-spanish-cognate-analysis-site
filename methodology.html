How the analysis was built

Methodology & data craftsmanship

The project blends academic datasets, ethical web scraping, statistical rigor, and transparent validation to deliver trustworthy insights.

Primary data sources

The analysis integrates multiple authoritative sources to capture true cognates, false friends, and bidirectional loanwords across English and Spanish.

CogNet Cognate Database

Academic dataset of cross-linguistic cognate relationships filtered for English-Spanish pairs and deduplicated across variants.

Etymonline & Oxford English Dictionary

Scraped earliest attestations and etymology details with respectful rate limiting, fallbacks, and ambiguity handling.

Google Ngrams API

Temporal frequency data for English and Spanish corpora, cleaned for OCR outliers and digitization bias.

False friends & loanword datasets

Curated false friend lists and a custom loanword master dataset capturing directionality and semantic shifts.

Data cleaning pipeline

  1. 01. Intelligent deduplication

    Normalized language pairs, resolved bidirectional duplicates, prioritized exact matches, and selected records with fullest metadata.

  2. 02. Text normalization

    Standardized casing, trimmed whitespace, handled diacritics carefully, and sanitized control characters.

  3. 03. Outlier detection

    Capped extreme frequency values at the 99th percentile, mitigating OCR and digitization anomalies.

  4. 04. LLM augmentation & validation

    Filled missing Spanish meanings via controlled LLM prompts, flagged synthetic entries, and spot-checked accuracy (88%).

Derived metrics

Similarity

  • Levenshtein distance with normalized similarity scores
  • Jaccard similarity over character bigrams

Complexity

  • Syllable count, word length, semantic polysemy
  • Frequency complexity using WordFreq Zipf scores

Temporal

  • First attestations from Etymonline and OED
  • Ngram-based emergence periods across six eras

Cultural domains

  • LLM-assisted domain classification across 21 categories
  • Manual review for ambiguous or multi-domain terms

Validation & quality checks

Sampling strategy

Performed stratified random sampling of 400 rows (95% confidence, ±5% margin) across relationship types.

Cross-source consistency

Correlated Ngram first appearances with Etymonline dates (r = 0.73) to confirm temporal reliability.

Data completeness

  • English word: 100%
  • Spanish word: 100%
  • Definitions: 69.8%
  • Cultural domains: 100%

Known limitations

Documented temporal, geographic, and register biases; called out semantic drift complexities and directionality nuances.
